Who I Am

Hello! I'm Abinash, a Data Science student with a growing passion for machine learning and AI. Interestingly, I stumbled into this field by chance. What started as simple curiosity turned into a fascinating journey when I saw how data could tell stories and solve problems in ways I never imagined.

My technical journey began with Python basics, then quickly expanded to libraries like NumPy and Pandas. The real turning point came when I discovered Andrew Ng's machine learning courses, which opened my eyes to the possibilities of AI. This led me to complete the Machine Learning Specialization from deeplearning.ai, followed by diving deeper with Andrej Karpathy's neural networks playlist. I've since earned both the Deep Learning and Natural Language Processing Specializations.

Along the way, I've built several projects that apply what I've learned – each one teaching me something new about implementing these technologies in practical ways. Currently, I'm on the lookout for internship opportunities where I can work on real-world challenges, apply my skills, and learn from experienced mentors. I believe there's so much more to discover in this field, and I'm excited to contribute to meaningful projects that make a difference.
